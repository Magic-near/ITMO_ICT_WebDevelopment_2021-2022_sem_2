API

/hotels/bookings/

Список бронирований

URL: /hotels/bookings/
Method: GET
Code: 200 OK
Response sample: 

{
  "count": 0,
  "next": "http://example.com",
  "previous": "http://example.com",
  "results": [
    {
      "id": 0,
      "check_in": "2019-08-24",
      "check_out": "2019-08-24",
      "room": 0,
      "client": 0
    }
  ]
}

Создание брони

URL: /hotels/bookings/create/
Method: POST
Code: 200 OK
Request sample: 

{
  "check_in": "2019-08-24",
  "check_out": "2019-08-24",
  "room": 0,
  "client": 0
}

Обновление/удаление брони

URL: /hotels/bookings/update/{id}/
Method: GET, PUT, PATCH, DELETE
Code: 200 OK
Response sample: 

{
  "id": 0,
  "check_in": "2019-08-24",
  "check_out": "2019-08-24",
  "room": 0,
  "client": 0
}
