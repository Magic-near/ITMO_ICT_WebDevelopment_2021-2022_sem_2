Введение

Цель:

Реализовать сайт для отеля, используя Django REST Framework для серверной части проекта и Vue.js — для клиентской.

Задачи:

Создать сайт со следующим функционалом:

  • авторизация и регистрация пользователей;
  • навигационная панель;
  • главная страница с описанием возможностей сервиса;
  • фильтры, триггеры, тесты, пагинация.

Описание сервиса

Данный сервис служит для упрощения процесса управления отелем. Он предоставляет возможности по

  • просмотру списка объектов,
  • просмотру одного объекта
  • добавлению нового объекта,
  • редактированию объекта,
  • удалению объекта,
  • фильтрации и поиску объектов

для моделей

  • комнат,
  • гостей,
  • бронирований,
  • сотрудников,
  • комментариев.
